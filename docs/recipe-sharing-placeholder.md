# ProDoughType — Recipe Sharing Placeholder

## Purpose
This file exists to preserve future ideas for portable dough sharing without expanding V1 scope.

## Core Idea
ProDoughType should eventually support portable dough recipes that can be:
- exported
- imported
- shared with other users
- loaded from community/forum sources

## Likely First Version
The first practical version should be simple file import/export.

Example flow:
1. User creates a dough
2. User exports a recipe file
3. Another user imports that file
4. ProDoughType loads the recipe into the standard step-by-step builder

## Future Possibilities
- forum/community import
- template exchange
- favorite/bookmarked shared recipes
- QR code transfer
- clone/tribute recipe packs

## Important Product Rule
Imported/shared recipes should never replace the builder.
They should only preload the builder and remain editable.

## Status
Placeholder only.
Do not prioritize until the core dough workflow is stable.
